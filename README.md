
# Autenticación Login

Full stack de Autenticación. Backend y front-end mediante Mongodb Atlas.


## Autor

- [@Michael Carrillo](https://www.github.com/OfficialMYKE)


## Despliegue
Copiar proyecto

```bash
  git clone https://link-to-project
```
Levantar Backend

```bash
  cd /backend
  npm run dev
```

Levantar Front-end

```bash
  cd /frontend
  npm run dev
```
